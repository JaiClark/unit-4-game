Pokemon Crystal Game


This is our Pokemon Crystal Game! The object of the game is to see if you can match the random number that will be given to you in the beginning of the game. So here are the rules.

There are four crystals below. Each crystal has a random value, and the value will be added up after every crystal click. So, all you have to do is press the crystals until you match the random number given and you win! Sounds easy right?

Each time you restart the game the value of the crystal will change and the random number will too. So good luck!
